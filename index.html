Sign In

Quick start

  • Open Login and sign in with one of the seeded accounts (student / instructor / admin).
  • Use Courses to add/update catalog entries (instructor/admin).
  • As a student, go to Register to search and enroll (seat checks and payment simulation included).
  • Instructors can enter grades on Grades; students can view them there as well.
  • Admin can view system summary on Admin.

Notes / Known rules enforced

  • Course code uniqueness is case-insensitive (prevents duplicates).
  • Students cannot register for the same course twice.
  • Student course limits: warning if < 4 courses; hard limit 7 courses maximum.
  • Payment simulation supports Visa (may fail to allow retry) and Cash (always succeeds).
Passwords for seeded accounts: student/studentpass, instructor/instructorpass, admin/adminpass
